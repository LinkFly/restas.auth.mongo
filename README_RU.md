RESTAS.AUTH.MONGO - Базированная на MONGODB аутентификация для основанных на RESTAS веб-приложений
------------------

См. пример использования в файле example.lisp

ВАЖНО! Система используется совместно с системой RESTAS.BASE.AUTH

* Для использования достаточно создать хранилище MONGO-STORAGE и установить его в текущем модуле:

```common-lisp
(restas.auth.base:set-storage (make-instance 'mongo-storage))
```

* При создании экземпляра MONGO-STORAGE, для корректного коннекта к СУБД MONGODB, нужно использовать
следующие ключевые аргументы:

 :hostname - хост на котором запущена MONGODB

 :port - порт, на котором MONGODB принимает клиентские запросы, если он отличен от порта по умолчанию

* А также можно указать дополнительные параметры для названия БД и названия коллекции с пользователями
(если нужно чтобы они отличались от значений по-умолчанию `*default-users-db*` и `*default-users-collection*`):

 :dbname - желаемое название БД

 :users-collection - желаемое название коллекции в которой будут сохраняться пользователи и хэши их паролей
